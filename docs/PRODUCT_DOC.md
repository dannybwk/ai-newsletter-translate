# AI Newsletter Translator — 產品文件

## 📌 目標
幫助用戶快速閱讀外文電子報，提供精準翻譯與重點摘要，同時保留版面設計。

## 🎯 功能
- 註冊 / 登入
- 專屬收件信箱
- 自動轉寄教學
- 收信翻譯 + 摘要
- 保留 HTML 排版
- Dashboard: 歷史紀錄、封數狀態
- 金流訂閱管理

## 🔗 技術
- 前端: Next.js + Tailwind
- 後端: Firebase Functions
- 資料庫: Firestore
- 收信: Gmail API
- 翻譯: OpenAI
- 金流: TapPay

## 💰 方案
- 免費試用: 2封/月
- Basic: $3.99/月 100封 單一語言
- Pro: $9.99/月 1000封 多語言
